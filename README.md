# Mis Ejercicios de Python - CECyT 9

Este repositorio contiene 10 programas desarrollados con la librería **NiceGUI**.

## 🛠️ Requisitos
Para ejecutar estos programas, necesitas instalar las dependencias:
`pip install -r requirements.txt`

## 🚀 Ejecución
El programa principal que conecta a todos los demás es:
`python main.py`

*Nota: Los archivos p1.py al p10.py son los módulos individuales de cada ejercicio.*
